INTRODUCTION
------------

The User Approval Dashboard module provides a dashboard where admins can approve
the users and can see a list of active and inactive users.

The User Export submodule provides an option to download user data in a
downloadable format such as csv,json,xml.

* For a full description of the module, visit the project page:
   https://www.drupal.org/project/user_approval_dashboard

* To submit bug reports and feature suggestions, or track changes:
   https://www.drupal.org/project/issues/user_approval_dashboard


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module. Visit
   https://www.drupal.org/node/1897420 for further information.


CONFIGURATION
-------------

The module has no menu or modifiable settings. There is no configuration. When
enabled, the module will provide a dashboard (from the toolbar, Dashboard) where
admin users can approve/decline users and view a list of active and inactive
users.
